## 1. Classic BPF Helpers (~212 total)

## 3. Kfunc Flags Reference

| Flag | Value | Meaning |
|------|-------|---------|
| `KF_ACQUIRE` | 1 << 0 | Returns acquired reference (must release) |
| `KF_RELEASE` | 1 << 1 | Releases a reference |
| `KF_RET_NULL` | 1 << 2 | May return NULL |
| `KF_KPTR_GET` | 1 << 3 | Gets kptr from map |
| `KF_DESTRUCTIVE` | 1 << 4 | Has destructive side effects |
| `KF_RCU` | 1 << 5 | Must be in RCU read section |
| `KF_IMPLICIT_ARGS` | 1 << 6 | Uses implicit context arguments |
| `KF_SLEEPABLE` | 1 << 7 | May sleep |
| `KF_ITER_NEW` | 1 << 8 | Iterator constructor |
| `KF_ITER_NEXT` | 1 << 9 | Iterator advance + return element |
| `KF_ITER_DESTROY` | 1 << 10 | Iterator destructor |
| `KF_RCU_PROTECTED` | 1 << 11 | Return value is RCU-protected |
| `KF_FASTCALL` | 1 << 12 | Optimized calling convention |
